Holochain is already in Alpha. You can start designing cutting edge distributed apps in JavaScript or Lisp using a test net today! The tools you build will be congruent with: customizable UI, the holochain DPKI app that allows identities to cross between application membranes, the Holo platform for connecting with users from the centralized web, and the potential to merge application environments with complete compatibility and no data loss. These capabilities are supported by the capacity to update validation requirements and application rules while offering agents a choice between new and old versions.
